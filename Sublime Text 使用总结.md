## 1.常用快捷键

|快捷键|功能|
|-|-|
|Alt + Shift + 1~9 (非小键盘)|屏幕显示相等数字的小窗口|
|Ctrl + d|自动选取当前单词|
|Ctrl + l|选取当前行|
|Ctrl + n|新建文件|
|Ctrl + p|定位|
|Ctrl + Shift+p|打开快捷键帮助面板|
|Ctrl + `|打开命令行|
|Ctrl + {|左缩进|
|Ctrl + }|右缩进|
